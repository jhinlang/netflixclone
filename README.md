#  Netflix UI Clone – Personal Project

This project is a **front-end UI clone of Netflix’s homepage**, built as a **personal and educational project**.  
Its goal is to replicate the overall layout, structure, and visual hierarchy of a real-world streaming platform using **core web technologies only**.

 This project is **not affiliated with Netflix, Inc.** and is not intended for commercial use.

---

## Why this project?

This project demonstrates my ability to:
- Build a **structured and semantic HTML5 layout**
- Reproduce a **complex, modern user interface**
- Organize front-end code in a clean and readable way
- Work with **CSS layouts (Flexbox, grids, alignment)**
- Pay attention to **UI/UX consistency**

It is part of my learning journey as a **front-end web developer**.

---

## Tech Stack

- **HTML5**
- **CSS3**
- **Font Awesome** (icons)
- No JavaScript framework
- No backend
